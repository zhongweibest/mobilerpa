# MobileRPA 系统使用说明书

## 1. 目的

本文档用于说明一台 Android 手机接入当前 MobileRPA 系统的最小准备步骤，重点覆盖真机基础设置、AutoJs6 权限准备和无线调试接入。

## 2. 手机准备

1. 打开手机“设置 -> 关于手机 -> 版本信息”。
2. 连续点击“版本号”，直到系统提示已开启开发者模式。
3. 安装 `AutoJs6`。

## 3. AutoJs6 权限准备

打开 `AutoJs6` 后，至少确认以下开关已经开启：

1. 无障碍服务。
2. 前台服务。
3. 忽略电池优化。
4. 悬浮窗或“显示在其他应用上层”权限。

另外还应在系统电池管理中，把 `AutoJs6` 加入白名单，避免被系统后台限制。

## 4. Android 开发者选项准备

1. 进入“设置 -> 开发者选项”。
2. 打开 `USB 调试`。
3. 打开 `无线调试`。
4. 进入“无线调试 -> 使用配对码配对设备”，记录当前显示的 IP、端口和配对码。

## 5. 中心端接入

1. 打开中心端管理后台的“设备发现”页面。
2. 使用“设备连接”或等价入口，输入上一步获得的手机 IP、端口和配对码。
3. 配对成功后，再执行设备发现、Agent 下发与 Agent 启动。

## 6. 当前说明边界

本文档只覆盖设备基础接入准备，不展开以下内容：

1. Agent 批量下发与启动细节。
2. 脚本上传、脚本下发与任务执行流程。
3. 工作流、计划任务和排障步骤。

这些内容分别以中心端、Agent 和脚本仓库中的对应文档为准。

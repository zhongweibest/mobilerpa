# 设备注册验收清单

## 验收目标
确认新设备能够首次注册，并在重复注册、中心重启后重新接入时保持正确身份。

## 前置条件
1. 中心服务已启动
2. SQLite 已初始化
3. 测试设备已安装 Agent
4. Agent 可访问中心服务

## 验收步骤
1. 清理测试设备本地配置，模拟首次安装
2. 启动 Agent
3. 观察 Agent 是否生成本地 `agent_uuid`
4. 观察 Agent 是否调用注册接口
5. 查看中心端是否创建设备记录
6. 查看返回的 `device_id` 是否被本地保存
7. 重启 Agent，再次启动
8. 验证是否仍识别为同一台设备
9. 重启中心服务
10. 再次验证设备是否能重新接入

## 预期结果
1. 首次启动生成唯一 `agent_uuid`
2. 注册成功返回唯一 `device_id`
3. 重启 Agent 不会生成新的 `device_id`
4. 重启中心后设备仍可恢复注册或接入

## 异常验证
1. 注册接口不可达时 Agent 是否有清晰错误日志
2. 重复注册时中心是否错误创建多条设备记录
3. 本地配置损坏时是否有可定位错误

## 验收结论
1. 通过
2. 有条件通过
3. 不通过

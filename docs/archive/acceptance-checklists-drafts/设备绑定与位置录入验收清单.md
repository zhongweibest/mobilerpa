# 设备绑定与位置录入验收清单

## 验收目标
确认待绑定设备可以被人工完成绑定、分组和物理位置录入。

## 前置条件
1. 新设备已注册
2. 后台可查看待绑定设备

## 验收步骤
1. 打开待绑定设备列表
2. 选择一台新设备
3. 录入 `device_name`
4. 录入 `physical_slot`
5. 选择或填写 `group_name`
6. 提交绑定
7. 返回设备总览页查看结果
8. 再次编辑位置或分组，验证可变更

## 预期结果
1. 新设备可以完成绑定
2. `device_id` 不变
3. `physical_slot` 可以维护
4. 设备在后台状态正确

## 异常验证
1. 重复录入相同 `physical_slot` 时是否有提示
2. 未填关键字段时是否阻止提交
3. 编辑失败时是否有明确反馈

## 验收结论
1. 通过
2. 有条件通过
3. 不通过
